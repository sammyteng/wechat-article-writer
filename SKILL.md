# 公众号文章写作 Skill

## 核心理念

- **写作引擎可替换**：正文写作模块默认使用通用骨架。用户指定「晚点风格」时，自动切换为 `latepost-writer` Skill 的六步写作流程。
- **一次触发，完整交付**：正文写完后，自动产出标题备选、CTA、朋友圈文案，并完成排版注入，直到发布 Checklist 生成为止。
- **不猜测，主动确认**：信息不足时优先向用户提问，不自行假设。

---

## 六步标准流程

### STEP 1：确认写文参数

主动询问用户以下信息（全部确认后再进入 STEP 2）：

```
请帮我确认以下写文参数：

1. 【主题】：这篇文章写关于什么？（公司/产品/行业/事件/人物）
2. 【写作风格】：通用模式 or 晚点LatePost深度报道模式？（默认：通用）
3. 【排版参考账号】：模仿哪个公众号的视觉风格？（默认：第101个机器人）
4. 【核心观点Thesis】：这篇文章最想说的一句话是什么？有独特角度吗？
5. 【目标字数】：长文（3000字+）/ 中文（1500字左右）/ 短评（1000字内）
```

参数收集完毕后，给用户一份确认摘要，等待「确认」后进入 STEP 2。

---

### STEP 2：选择内容骨架

根据 STEP 1 参数，从以下骨架库中匹配：

| 骨架类型 | 适用场景 | 资源文件 |
|---------|---------|---------|
| 战略拆解型 | 公司发展路径/品牌战略全解读 | `resources/playbook_strategy.md` |
| 人物特写型 | 创始人/高管深度人物稿 | `resources/playbook_profile.md` |
| 行业变局型 | 赛道趋势/市场格局分析 | `resources/playbook_industry.md` |
| 产品逻辑型 | 产品深度拆解/用户体验分析 | `resources/playbook_product.md` |
| 创投生态型 | 融资/资本/投资逻辑叙事 | `resources/playbook_capital.md` |

输出：推荐骨架 + 说明理由，等用户确认。

> **若用户选择晚点风格**：从 STEP 3 起，调用 `latepost-writer` Skill 的写作逻辑（STEP 3 采集、STEP 4 写作），完成后回到本 Skill 的 STEP 4（内容包）继续执行。

---

### STEP 3：全网素材采集 + 正文写作

**素材采集（至少覆盖以下渠道）**：
- 官方渠道：官网、公告、投资者关系页
- 权威媒体：36氪、虎嗅、界面新闻、财经、晚点LatePost
- 数据报告：IDC、艾瑞、欧睿、招股书/财报
- 社区讨论：知乎、小红书、微博
- 海外视角：英文媒体、Twitter/X、Reddit

**正文写作规范**（通用模式）：
- 按选定骨架结构展开
- 无首行缩进，段落短小（手机端友好）
- 数字序号小标题：`01 标题内容`
- 关键句加粗强调
- 数据必须有来源支撑，不捏造数据
- 尾部：`-end-` + `文 | [账号名]`

---

### STEP 4：生成配套内容包

正文完成后，**立即输出**以下配套内容，无需用户再次触发：

#### 📌 标题备选（3个）

分别对应三种不同的读者钩子策略：

```
【悬念型】（制造认知冲突，让人想点进来）
→ 示例：追觅，为什么主动放弃价格战？

【数据型】（信息密度高，可信度强）
→ 示例：150亿营收、30国第一，追觅用8年做成了什么

【情绪型】（共鸣/惊叹/忧虑，触发情感反应）
→ 示例：追觅跑得太快了，快到有点让人不安
```

#### 📌 CTA 话术

置于文章结尾，2-3句，引导关注/转发/留言：

```
写法参考：
- 语气与账号风格一致（第101个机器人：理性+科技感+有温度）
- 提出一个思考问题引导留言
- 结尾引导关注或转发
```

#### 📌 朋友圈转发文案

50-80字，可直接复制使用：

```
写法要求：
- 第一句提炼核心观点（带数据或强结论）
- 第二句说明为什么值得读
- 结尾：「👆 点击阅读全文」或「链接在评论区」
```

---

### STEP 5：排版交付

1. 将正文 Markdown 通过 JavaScript 注入 `https://md.doocs.org`
2. 截图确认右侧渲染效果
3. 通知用户：

```
✅ 排版已就绪！

你的浏览器中已打开 md.doocs.org，文章内容已加载。

操作步骤：
1. 打开浏览器 → 找到 md.doocs.org 标签页
2. 点击右上角「复制」按钮
3. 进入微信公众号后台 → 新建图文消息 → Ctrl+V 粘贴
4. 手动将 01～0X 小标题颜色改为橙色（#FF6600）
```

---

### STEP 6：发布前 Checklist

输出标准化发布确认清单：

```
📋 发布前检查清单

内容部分：
□ 从3个标题中选定1个，填入公众号标题栏
□ 摘要/作者等基本信息已填写
□ 正文已粘贴完整

排版部分：
□ 01～0X 小标题颜色已改为橙色
□ 封面图已设置（建议尺寸：900×383px）
□ 分组/标签已选择

配套物料：
□ CTA 话术已添加至文章末尾
□ 朋友圈文案已复制备用

发布设置：
□ 发布时间已确认（建议：工作日早8-9点 or 晚8-10点）
□ 预览已在手机端查看一遍
```
