# 骨架二：人物特写型

适用场景：创始人、CEO、投资人或关键决策者的深度人物稿。

---

## 结构模板

```
01  [一个定义这个人的细节]

用一个具体的行为细节或场景，建立读者对这个人的第一印象。
- 不从简历开始，从一个"反常识"的选择或习惯开始
- 这个细节要指向他/她的核心性格特质

02  [原点：他/她为什么走上这条路]

追溯形成关键性格/判断力的起点事件。
- 有哪段经历是现在决策风格的来源？
- 避免"从小就有梦想"的老套路，找具体事件

03  [决定性时刻]

讲述一次最能体现此人决策风格的关键选择。
- 当时面临什么压力或困境？
- 他/她选择了什么？放弃了什么？
- 这个选择的代价和结果

04  [争议与批评]

平衡视角，呈现外界对这个人的质疑或不同评价。
- 离职员工/竞争对手/外部观察者怎么看？
- 不回避矛盾，客观呈现

05  [这个人眼中的下一个赌注]

聚焦当下：他/她现在最在意的事是什么？
- 用他/她自己的话（引语）
- 或用行为来说明（"他/她在意的是……"而非"他/她说……"）
```

---

## 写作要点

- 人物必须立体，不只是成功者光环
- 用对话/场景代替形容词
- 避免"他/她认为""他/她表示"等空洞词，用具体行为和决定说明人物
