# 骨架一：战略拆解型

适用场景：解读一家公司/品牌的发展路径、战略转折、核心决策。

---

## 结构模板

```
01  [核心技术/产品] 是命门

说明公司最底层的竞争壁垒，用具体的数字和对比建立读者认知。
- 这项核心能力是什么？为什么别人难以复制？
- 用一个让普通人能理解的类比说明

02  [关键转折点] 的故事

讲述公司战略中最关键的一次"选择"或"转折"。
- 当时的市场环境是什么？
- 公司做了哪个不寻常的决定？
- 结果如何？

03  [全球化/品类扩张/渠道变革] 中的反直觉打法

揭示公司增长策略中与行业惯例相反的选择。
- 其他人怎么做的？
- 这家公司选择了什么不同的路？
- 数据支撑：这个打法的结果

04  [当前最大赌注]

描述公司当下正在进行的最大战略押注。
- 规模/投入/时间线是什么？
- 这个赌注成功的条件是什么？
- 失败的风险是什么？

05  这个故事的悬念在哪里

不给确定性结论，而是提出核心矛盾或未解之谜：
- 市场、对手、资金、组织的哪个维度是最大变量？
- 留白：等市场给出答案
```

---

## 写作要点

- 所有判断必须有数据或事实支撑
- 避免空泛的"颠覆性""领先""引领"等修饰词
- 结尾不下终局判断，保持开放性
